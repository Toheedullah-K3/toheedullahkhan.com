# NotesBuddy

A comprehensive study platform with notes, flashcards, quizzes, AI chatbot, and interactive learning tools.

## Overview

NotesBuddy is an all-in-one learning and study management platform designed to help students and learners master their subjects efficiently. Combining traditional note-taking with modern educational tools like flashcards, quizzes, and AI-powered chatbot assistance, NotesBuddy creates an interactive learning experience tailored to your needs.

## Key Features

- **Notes Management**: Create, organize, and manage study notes with rich text formatting
- **Flashcards**: Build customizable flashcard decks for spaced repetition learning
- **Interactive Quizzes**: Test your knowledge with auto-generated or custom quizzes
- **AI Chatbot**: Get instant help and explanations from an intelligent learning assistant
- **Study Progress Tracking**: Monitor your learning progress and identify weak areas
- **Interactive Learning Tools**: Engage with dynamic content to enhance retention
- **Multi-format Support**: Support for various content formats including MDX for rich content

## Technologies Used

- **Frontend**: Next.js with React for a modern, fast web application
- **Language**: TypeScript for type-safe, maintainable code
- **Database**: MongoDB for flexible data storage
- **Styling**: Tailwind CSS for responsive, utility-first design
- **UI Components**: shadcn/ui for polished, accessible components
- **Deployment**: Vercel for seamless, production-ready hosting
- **Content**: MDX support for rich, interactive documentation and content

## Project Highlights

- Built with Next.js for optimal performance and SEO
- TypeScript for robust, maintainable codebase
- Responsive design that works across all devices
- Integration with AI-powered learning assistants
- Comprehensive study tools in one platform
- Beautiful UI built with shadcn/ui components
- Deployed on Vercel for reliability and speed
