# SentiScope

Advanced sentiment analysis platform to analyze, compare, and visualize sentiment trends across multiple topics, platforms, and AI models in real time.

## Overview

SentiScope is a comprehensive sentiment analysis application designed to help users understand and track emotional trends in text data. Whether you're monitoring social media sentiment, analyzing customer feedback, or researching public opinion on specific topics, SentiScope provides powerful tools to extract, compare, and visualize sentiment insights in real-time.

## Key Features

- **Real-time Sentiment Analysis**: Analyze sentiment trends across multiple topics simultaneously with live updates
- **Multi-Platform Support**: Compare sentiment data from various sources and platforms
- **AI Model Comparison**: Leverage multiple AI models to cross-validate sentiment predictions
- **Interactive Visualizations**: Beautiful, responsive charts and graphs to track sentiment over time
- **Topic Tracking**: Monitor sentiment trends for multiple topics side-by-side
- **Detailed Analytics**: Dive deep into sentiment metrics, breakdowns, and historical data

## Technologies Used

- **Frontend**: React for interactive user interfaces
- **Backend**: Node.js with Express for robust API services
- **Database**: MongoDB for flexible, scalable data storage
- **Styling**: Tailwind CSS for modern, responsive design

## Project Highlights

- Built with React for a responsive and interactive frontend experience
- Scalable Node.js and Express backend architecture
- MongoDB database for efficient data management
- Tailwind CSS for a clean, modern user interface
- Real-time sentiment analysis and visualization capabilities
